# 대학 교양글쓰기 교재 구조 설계안

## 프로젝트 개요
- **목적**: 2단계 심화 트랙 교재 개발 (교양글쓰기 → 학술글쓰기)
- **대상**:
  - 1단계: 인문/공학 계열 전체 (교양글쓰기)
  - 2단계: 인문계열 심화학습 (학술글쓰기)
- **핵심 과제**: 두 단계 간 난이도 조절 및 학습 연계성 확보

---

## 국내 대학 교재 조사 결과

### 주요 대학 교양글쓰기 교재 구조

#### 1. 고려대학교 『대학 글쓰기의 이해』
**3장 구조의 과정 중심 접근**
- **제1장**: 글쓰기의 이해 (개념, 목적, 윤리)
- **제2장**: 글쓰기의 과정
  - 계획하기 (목적/독자 확인, 아이디어 생성, 개요 구성)
  - 집필하기 (어휘/문장 생성, 단락 구성, 서론/결론 완성)
  - 수정하기
- **제3장**: 글쓰기의 방법과 유형
  - 서술방법: 비교/대조, 분류, 서사/묘사, 인과, 논증
  - 글쓰기 유형: 창의적, 설득적, 학술적 글쓰기
- **부록**: 발표와 토론, 언어 사용 규범

**출처**: [고려대학교출판문화원](https://kupress.com/books/8139/)

#### 2. 『대학 글쓰기 1』
**4부 구조의 체계적 접근**
- **제1부**: 대학 생활과 글쓰기 (의의, 목표, 윤리)
- **제2부**: 글쓰기의 과정 (주제 잡기, 자료 수집, 구성, 문장 쓰기, 글 다듬기)
- **제3부**: 글쓰기의 방법 (정의, 비교, 분류, 서사, 묘사, 논증)
- **제4부**: 글쓰기의 실제

**출처**: [대학 글쓰기 1](https://sky.kipa.re.kr/library/10110/contents/7040174)

### 학술글쓰기 교재 구조 특징

#### 핵심 요소
1. **학술논문의 개념과 특성**
   - 객관성, 검증가능성, 정확성
   - 논증의 엄밀성, 소통의 용이성
   - 연구 윤리 및 글쓰기 윤리

2. **연구방법론**
   - 주제 선정과 연구 문제 설정
   - 선행연구 검토
   - 연구 설계 및 자료 수집

3. **학술적 글쓰기 기술**
   - 인용과 출처 표기법 (APA, MLA, Chicago 등)
   - 논문 구성 (서론, 본론, 결론)
   - 참고문헌 작성

**출처**:
- [경희대학교 학술논문 작성법](https://libguides.khu.ac.kr/global/writing/03)
- [서울대학교 온라인 글쓰기교실](https://owl.snu.ac.kr/2465/)
- [대학 학술적 글쓰기 인용 출처 작성법 교육](https://www.kci.go.kr/kciportal/landing/article.kci?arti_id=ART003031569)

### 난이도 체계 연구 결과

#### 고려대학교 2단계 시스템
- **기초 글쓰기**: "글쓰기 기초"를 훈련하여 비판적 사고력과 자기표현 능력 개발
  - 학습자가 자신을 성찰하고 학업 수행에 필요한 글쓰기 특성 이해
  - 비판적 관점에서 학술적 글쓰기 수행

- **심화 글쓰기**: 학술 논문 작성 과정을 체계적으로 학습
  - 고급 학술적 글쓰기 능력 개발
  - 실제 학술 논문 작성 훈련

**출처**: [고려대학교 심화과정](https://univ.korea.ac.kr/uc/curriculum/requirements_writing_advanced.do)

#### 단계별 글쓰기 시스템 모듈화
- **1-2단계**: 글쓰기 센터 중심 "Learning to write" (쓰기를 배우기)
- **3-4단계**: 교양/전공 중심 "Writing to learn" (쓰기로 배우기)
- 사전 평가, 학년, 전공, 수준별 환경 고려한 모듈 구성

**출처**: [학습자 수준을 고려한 교육 과정](https://www.kci.go.kr/kciportal/ci/sereArticleSearch/ciSereArtiView.kci?sereArticleSearchBean.artiId=ART001313851)

## 2단계 심화 트랙 교재 구조 설계

### 설계 원칙
1. **연계성**: 1단계에서 학습한 내용을 2단계에서 심화·확장
2. **난이도 조절**: 점진적 난이도 상승 (기초 → 심화)
3. **실용성**: 각 계열 특성에 맞는 실제 사례 및 실습 제공
4. **체계성**: 과정 중심 접근과 결과 중심 접근의 균형

---

## 1단계: 교양글쓰기 교재 구조 (인문/공학 계열 공통)

### 목표
- 대학 학습에 필요한 기초적 글쓰기 능력 함양
- 비판적 사고와 논리적 표현 능력 개발
- 다양한 글쓰기 유형에 대한 이해와 실습

### 난이도: **기초~중급**

---

### 제1부: 대학 글쓰기의 이해 (2주)

#### 1장. 글쓰기란 무엇인가
- 1.1 대학 글쓰기의 의의
- 1.2 사고와 표현
- 1.3 독자와 목적

#### 2장. 대학생과 글쓰기
- 2.1 학문 공동체와 글쓰기
- 2.2 비판적 사고와 논리적 표현
- 2.3 글쓰기 윤리와 표절

**학습 목표**
- 대학 글쓰기의 특성과 중요성 이해
- 학술적 글쓰기의 기본 윤리 습득
- 비판적 사고의 중요성 인식

**난이도 조절**
- 개념 중심의 이론 학습
- 친숙한 사례와 예시 활용
- 계열 구분 없이 공통 내용

---

### 제2부: 글쓰기의 과정 (6주)

#### 3장. 계획하기
- 3.1 주제 선정과 범위 설정
- 3.2 독자 분석과 목적 설정
- 3.3 브레인스토밍과 아이디어 생성
- 3.4 개요 작성하기

#### 4장. 집필하기
- 4.1 문장의 기본 원리
  - 명확성, 간결성, 정확성
  - 주술 호응, 수식 관계
- 4.2 단락의 구성
  - 주제문과 뒷받침 문장
  - 단락의 통일성과 일관성
- 4.3 서론 쓰기
  - 주제 제시와 흥미 유발
  - 글의 방향 제시
- 4.4 본론 전개하기
  - 논리적 전개 방법
  - 근거와 사례 제시
- 4.5 결론 작성하기
  - 내용 요약과 의미 부여
  - 제목 정하기

#### 5장. 수정하기
- 5.1 내용 수정
  - 논리성과 타당성 검토
  - 근거의 적절성 평가
- 5.2 표현 수정
  - 문장과 단락 다듬기
  - 어휘 선택과 문체 통일
- 5.3 형식 수정
  - 맞춤법과 띄어쓰기
  - 인용과 각주 점검

**학습 목표**
- 체계적인 글쓰기 과정 습득
- 단계별 글쓰기 전략 적용 능력
- 자기 수정 및 동료 피드백 능력 개발

**난이도 조절**
- 단계별 실습 활동 포함
- 계열별 예시 제공 (인문/공학)
- 워크숍 및 첨삭 지도 병행

---

### 제3부: 글쓰기의 방법 (4주)

#### 6장. 서술 방법의 이해
- 6.1 정의와 설명
- 6.2 비교와 대조
- 6.3 분류와 구분
- 6.4 인과 관계
- 6.5 서사와 묘사

#### 7장. 논증의 원리
- 7.1 주장과 근거
- 7.2 논리적 오류
- 7.3 설득 전략
- 7.4 반론 제기와 재반론

**학습 목표**
- 다양한 서술 방법 이해와 활용
- 논리적 논증 능력 개발
- 설득적 글쓰기 전략 습득

**난이도 조절**
- 방법론 중심의 이론 학습
- 계열별 맞춤 예시 (인문: 사회 현상 분석 / 공학: 기술 설명)
- 점진적 난이도 상승

---

### 제4부: 글쓰기의 실제 (3주)

#### 8장. 비평적 글쓰기
- 8.1 서평 쓰기
- 8.2 영화/공연 비평
- 8.3 비판적 분석문

#### 9장. 설득적 글쓰기
- 9.1 주장하는 글
- 9.2 문제 해결형 글쓰기
- 9.3 제안서 작성

#### 10장. 학술적 글쓰기 입문
- 10.1 보고서의 기본 형식
- 10.2 자료 조사와 활용
- 10.3 인용과 출처 표기 기초
- 10.4 간단한 연구 보고서 작성

**학습 목표**
- 실제 글쓰기 과제 수행 능력
- 다양한 글쓰기 유형 경험
- 학술적 글쓰기로의 전환 준비

**난이도 조절**
- 계열별 차별화된 과제 (인문: 서평, 비평 / 공학: 기술 보고서, 제안서)
- 학술적 글쓰기는 입문 수준으로 제한
- 2단계로의 연결고리 제공

---

### 부록
- A. 한글 맞춤법과 표준어 규정
- B. 문장 부호 사용법
- C. 발표와 토론의 기초
- D. 참고문헌 작성법 (기초)

## 2단계: 학술글쓰기 교재 구조 (인문계열 심화)

### 목표
- 학술 논문 작성 능력 개발
- 연구 방법론의 이해와 적용
- 학문 공동체의 담화 관습 습득

### 난이도: **중급~고급**

---

### 제1부: 학술적 글쓰기의 이해 (2주)

#### 1장. 학술적 글쓰기란 무엇인가
- 1.1 학술적 글쓰기의 개념과 특성
  - 객관성과 검증가능성
  - 논증의 엄밀성
  - 학문적 담화 공동체
- 1.2 학술적 글쓰기 vs 교양적 글쓰기
  - 목적과 독자의 차이
  - 형식과 문체의 차이
  - 근거 제시 방식의 차이
- 1.3 학문 분야별 글쓰기 특성
  - 인문학 분야의 글쓰기
  - 사회과학 분야의 글쓰기
  - (참고) 자연과학/공학 분야의 글쓰기

#### 2장. 연구 윤리와 학술적 글쓰기 윤리
- 2.1 연구 윤리의 중요성
- 2.2 표절과 자기 표절
- 2.3 인용의 윤리
- 2.4 데이터 조작과 위조
- 2.5 저작권과 연구 윤리

**학습 목표**
- 학술적 글쓰기의 본질 이해
- 1단계 교양글쓰기와의 차별성 인식
- 연구 윤리 의식 확립

**난이도 조절**
- 1단계의 "글쓰기 윤리"에서 "연구 윤리"로 심화
- 학문 분야별 사례 제시
- 이론과 실제 사례 연계

---

### 제2부: 연구 설계와 자료 수집 (4주)

#### 3장. 연구 주제 선정과 문제 설정
- 3.1 연구 주제 탐색하기
  - 학문적 관심사 발견
  - 선행 연구 검토
  - 연구 가능성 평가
- 3.2 연구 문제 설정하기
  - 연구 질문 구체화
  - 연구 목적과 필요성
  - 연구 범위 설정
- 3.3 연구 가설 수립 (사회과학)
- 3.4 연구 계획서 작성

#### 4장. 선행 연구 검토
- 4.1 학술 자료의 종류와 특성
  - 학술지 논문, 학위논문, 학술서
  - 1차 자료와 2차 자료
- 4.2 문헌 검색 방법
  - 학술 데이터베이스 활용 (RISS, KISS, DBpia 등)
  - 검색어 설정 전략
  - 해외 데이터베이스 (JSTOR, Google Scholar 등)
- 4.3 선행 연구 분석과 정리
  - 비판적 읽기
  - 연구 동향 파악
  - 연구 공백 발견
- 4.4 문헌 정리 도구 활용
  - EndNote, Zotero 등 서지관리 프로그램

#### 5장. 연구 방법론 (인문계열 중심)
- 5.1 질적 연구 방법
  - 텍스트 분석
  - 담론 분석
  - 내용 분석
  - 사례 연구
- 5.2 양적 연구 방법 기초 (사회과학)
  - 설문 조사
  - 통계 분석 개요
- 5.3 자료 수집과 정리
  - 1차 자료 수집
  - 2차 자료 활용
  - 자료의 신뢰성과 타당성

**학습 목표**
- 독자적인 연구 주제 설정 능력
- 체계적인 문헌 검토 능력
- 연구 방법론의 이해와 선택 능력

**난이도 조절**
- 1단계의 "자료 조사"에서 "학술 자료 검색"으로 심화
- 실제 데이터베이스 실습 포함
- 인문계열 중심이되 사회과학 방법론도 소개

---

### 제3부: 학술 논문 작성 (6주)

#### 6장. 논문의 구조와 구성
- 6.1 학술 논문의 기본 구조
  - IMRaD 구조 (사회과학)
  - 인문학 논문의 구조
- 6.2 제목 작성하기
  - 명확하고 구체적인 제목
  - 주요 키워드 포함
- 6.3 초록 작성하기
  - 초록의 기능과 유형
  - 효과적인 초록 작성법

#### 7장. 서론 작성하기
- 7.1 연구 배경과 필요성
- 7.2 선행 연구 검토와 연구 공백
- 7.3 연구 문제와 목적
- 7.4 연구 방법과 범위
- 7.5 논문의 구성 소개

#### 8장. 본론 전개하기
- 8.1 논리적 구조 설계
  - 장·절 구성의 원리
  - 논증의 전개 방식
- 8.2 주장과 근거의 제시
  - 이론적 근거
  - 경험적 근거
  - 사례 분석
- 8.3 학술적 문체
  - 객관적이고 명확한 문장
  - 학술 용어의 정확한 사용
  - 논리적 연결어 활용
- 8.4 표와 그림의 활용
  - 시각 자료의 효과적 제시
  - 캡션 작성법

#### 9장. 결론 작성하기
- 9.1 연구 결과 요약
- 9.2 연구의 의의와 기여
- 9.3 연구의 한계
- 9.4 향후 연구 방향 제시

**학습 목표**

**난이도 조절**

- 실제 학술지 논문 분석 실습
- 단계별 쓰기 과제 (서론 → 본론 → 결론)

- 10.1 인용의 목적과 기능
- 10.2 직접 인용과 간접 인용
- 10.3 인용의 통합 (Integration)
  - 문장 내 인용
  - 블록 인용

- 10.5 인용 시 주의사항
  - 과다 인용 피하기
  - 맥락에 맞는 인용

#### 11장. 각주와 미주
- 11.1 각주/미주의 기능

- 11.2 각주 작성 방식
  - Chicago Style (Notes-Bibliography)

#### 12장. 참고문헌 작성
- 12.1 참고문헌의 중요성
- 12.2 주요 인용 스타일
  - **APA 7th** (사회과학, 교육학)

  - **Chicago 17th** (역사학, 예술)
  - **한국어 논문 작성 표준 양식**
- 12.3 자료 유형별 참고문헌 작성

  - 편저서 내 장/논문
  - 온라인 자료, 신문 기사
  - 영상 자료, 인터뷰
- 12.4 서지관리 프로그램 활용
  - EndNote, Zotero, Mendeley

**학습 목표**
- 정확한 인용 기법 습득
- 분야별 인용 스타일 이해와 적용
- 윤리적인 학술적 글쓰기 실천

**난이도 조절**
- 1단계의 "기초적 인용"에서 "체계적 인용 시스템"으로 심화

- 실제 논문 작성 과제에 적용

## 난이도 조절 전략

### 1단계 → 2단계 연계 방안

#### 개념의 확장
- **1단계**: 글쓰기 윤리, 표절 방지
- **2단계**: 연구 윤리, 학술적 정직성, IRB

#### 기술의 심화
- **1단계**: 자료 조사와 활용
- **2단계**: 학술 데이터베이스 검색, 선행 연구 비판적 검토

#### 형식의 고도화
- **1단계**: 기본적 보고서 형식, 간단한 인용
- **2단계**: IMRaD 구조, 체계적 인용 시스템 (APA, MLA)

#### 독자의 전문화
- **1단계**: 일반 독자 대상 설명적 글쓰기
- **2단계**: 학술 공동체 대상 논증적 글쓰기

## 계열별 차별화 전략

### 1단계 교양글쓰기: 계열별 예시 및 과제

| 내용 | 인문계열 | 공학계열 |
|------|---------|---------|
| **서술 방법** | 문학 작품 분석, 역사적 사건 설명 | 기술 프로세스 설명, 시스템 분석 |
| **논증** | 사회 현상 비판, 가치 판단 | 문제 해결 방안, 기술적 타당성 |
| **과제 예시** | 서평, 영화 비평, 사회 이슈 분석 | 기술 보고서, 실험 보고서, 제안서 |
| **자료** | 문학 텍스트, 인문 서적 | 기술 매뉴얼, 공학 논문 |

### 2단계 학술글쓰기: 인문계열 중심 (사회과학 포함)

- **인문학**: 텍스트 분석, 문헌 연구, 해석학적 접근
- **사회과학**: 양적/질적 연구, 실증적 분석, 이론 적용
- **공학 계열은 별도 전공 글쓰기 과정으로 분리** (본 교재 범위 외)

## 평가 방안

### 1단계: 교양글쓰기
- **출석 및 참여** (10%): 워크숍, 동료 평가 참여
- **단계별 과제** (40%): 계획서, 초고, 수정본
- **중간 과제** (20%): 비평문 또는 설득문
- **기말 과제** (30%): 학술적 보고서 (3,000-5,000자)

### 2단계: 학술글쓰기
- **출석 및 참여** (10%): 세미나, 동료 검토 참여
- **연구 계획서** (20%): 주제, 문제, 방법론
- **선행 연구 검토** (20%): 문헌 분석 보고서
- **최종 논문** (50%): 학술 논문 형식 (8,000-15,000자)
  - 구조 및 논리 (20%)
  - 내용 및 논증 (20%)
  - 인용 및 형식 (10%)
